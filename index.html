▼ レポートの出し方（MT5 / MT4）

MT5の場合

  1. MT5下部の「ツールボックス」を開く（表示されていない場合は Ctrl+T）
  2. 「口座履歴」タブをクリック
  3. 表の上で右クリック →「期間」→「全履歴」を選ぶ
  4. もう一度右クリック →「レポート」→ HTML 形式で保存
  5. 保存されたファイルを、この画面にドラッグ＆ドロップ

MT4の場合

  1. ターミナル（Ctrl+T）の「口座履歴」タブを開く
  2. 表の上で右クリック →「期間のカスタム設定」で全期間にする
  3. 右クリック →「レポートの保存（詳細）」でHTML保存
  4. 保存されたファイルを、この画面にドラッグ＆ドロップ

KURAMAシリーズEAをお使いの場合：MT5の「ファイル」→「データフォルダを開く」→ MQL5 → Files 内にある取引CSV（例：KGA_EA_○○○.csv）をそのまま読み込むこともできます。

うまく読み込めない場合は、レポートの形式（HTML）と期間設定をご確認ください。

累積損益カーブ

左から右へ、決済のたびに損益を積み上げた軌跡。金色の線が右肩上がりなら資産が積み上がっています。赤い網掛けは「直近ピークからの下落（ドローダウン）」区間。カーソルを乗せると日付と金額が出ます。

月別損益

月ごとの合計。緑の月が多いほど安定して積めています。

曜日別損益

決済した曜日ベースの合計。特定の曜日に負けが偏っていないかが分かります。

時間帯別損益

エントリー時刻（レポート記載の時刻基準）ごとの合計。どの時間帯が稼ぎ頭かが分かります。

1取引あたりの損益分布

横軸が1取引の損益、縦軸が回数。右側（緑）に山があれば勝ちが太く、左端に大きな棒があるなら大負けに注意。

BUY / SELL 別

方向ごとの成績。どちらかに偏って負けている場合、相場つき（トレンド方向）との相性が見えます。
